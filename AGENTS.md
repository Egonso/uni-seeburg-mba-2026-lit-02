# Agent Instructions

Dieses Repo ist temporär öffentlich. Behandle es als Public-Bootstrap, nicht als vollständiges privates Arbeitsrepo.

## Harte Regeln

- Keine personenbezogenen Daten hinzufügen.
- Keine E-Mails, Formulare, Audios, Videos, Transkripte oder Rohdaten committen.
- Keine vollständigen Lernplattform-PDFs oder Fremdtexte committen.
- Keine Quellen erfinden.
- KI-Output ist keine Quelle.
- Immer echte Umlaute schreiben: ä, ö, ü.

## Projekt-Skill

Vor inhaltlicher Arbeit `skills/uni-seeburg-mba-lit-02/SKILL.md` lesen und dessen Gates anwenden. Wenn ein Codex-Client lokale Skills unterstützt, diesen Skill für alle Forschungsfragen-, Recherche-, Outline-, Quellen-, Framework-, Review- und Drafting-Aufgaben in diesem Repo aktivieren.

## Arbeitsreihenfolge

1. Veröffentlichungsgrenze lesen.
2. 12-Schritte- und Outline-Gate lesen.
3. Forschungsfrage und `OUTLINE.md` stabilisieren.
4. Bewertungsmapping aktualisieren.
5. Erst dann Methode, Quellenmatrix oder Drafts vertiefen.

Kein Drafting ohne tragfähige Outline.
