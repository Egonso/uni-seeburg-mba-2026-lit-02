# Uni Seeburg MBA 2026 LIT-02

Temporär öffentliches Bootstrap-Repo für eine theoretische bzw. Literaturarbeit. Die genaue Forschungsfrage muss vor tieferem Drafting finalisiert werden.

Dieses Repo enthält absichtlich keine personenbezogenen Daten, keinen konkreten Arbeitstitel, keine privaten Mails, keine Formulare, keine Lernplattform-Volltexte und keine Rohdaten. Es ist nur die öffentliche Arbeitsstruktur für Agenten.

## Arbeitslogik

1. `docs/00-publication-boundary.md` lesen.
2. `docs/01-12-schritte-outline-gate.md` lesen.
3. `workspace/01_expose_und_forschungsfrage/FORSCHUNGSFRAGEN.md` finalisieren.
4. Erst danach Literaturmatrix, Syntheselogik und Framework-Entwicklung vertiefen.

## Aktueller Typ

- Arbeitstyp: Literaturarbeit / konzeptionelles Framework.
- Primärer Qualitätsfokus: Forschungsfrage schärfen, Reviewlogik, Quellenqualität, Framework-Eigenleistung, Praxisimplikationen.
- Public-Grenze: konkrete Person, Thema, Unternehmensdaten und interne Praxisunterlagen bleiben außerhalb dieses öffentlichen Repos.

## Wichtig

Nach Umstellung auf private Sichtbarkeit kann der private Agentenkontext importiert werden. Bis dahin nur Platzhalter und generische Struktur verwenden.
